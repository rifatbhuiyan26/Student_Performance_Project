Project Overview

This project analyzes the Students Performance Dataset using Python, Pandas, NumPy, and Matplotlib. The main goal is to understand how gender, parental education level, lunch type, and test preparation course affect student performance.

Dataset Information

  • Dataset Name: Students Performance Dataset
  • Rows: 1000
  • Columns: 8
  • Source: Kaggle
